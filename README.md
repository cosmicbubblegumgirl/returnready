# ReturnReady

ReturnReady is a standalone reverse logistics product website.

- Created as **A Quantum Cupcake Creation (SimonĂŠ Govender)**.
- Pages: control tower, shopper portal, warehouse intake, resolution lab, database, login, and case study.
- Data: 100 preloaded return records in `api/data.json`.
- Demo login: `simone@returnready.demo` / `demo123`.
- Static backend for GitHub Pages: `api/data.json` and `api/users.json`.
- Local backend: run `npm start` to use `server/server.js` with `/api/health`, `/api/login`, and `/api/records`.
- Interactions: theme toggle, demo login, return route simulator, shopper stepper, warehouse scan board, searchable database, JSON export, and local CRUD-style records.
